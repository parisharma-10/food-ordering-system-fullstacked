# Food Ordering System (Full Stack)

A full-stack food ordering application built using **React (Frontend)** and **Spring Boot (Backend)** with REST APIs.
a basic interface system for begginers learning to connect frontend & backend with axios 

---

## 🚀 Features

* Browse food items
* View menu dynamically from backend
* REST API integration
* Full-stack architecture

---

## 🛠 Tech Stack

* **Frontend:** React, Axios, CSS
* **Backend:** Spring Boot, Java
* **Database:** MySQL
* **Tools:** Maven, Git

---

## 📂 Project Structure

```
food-ordering-system-fullstacked/
 ├── food-backend/    # Spring Boot backend
 ├── food-frontend/   # React frontend

```

---

## ▶️ How to Run Locally

### 🔹 Backend

```
cd food-backend
mvn spring-boot:run
```

Runs on: http://localhost:8080

---

### 🔹 Frontend

```
cd food-frontend
npm install
npm start
```

Runs on: http://localhost:3000 or 3001

---

## 🔗 API Endpoint

```
GET http://localhost:8080/api/food
```
